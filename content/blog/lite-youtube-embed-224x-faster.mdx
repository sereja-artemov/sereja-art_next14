## Минусы стандартного способа через iframe
Стандартные embed-скрипты для YouTube очень тяжелые и&nbsp;весят сотни килобайт. Весь этот ком скриптов загружается сразу и&nbsp;сильно тормозит загрузку страницы, из-за этого снижается оценка в&nbsp;Google PageSpeed Insights. В&nbsp;итоге пользователь расстроен, вы&nbsp;расстроены, грусть, печаль, ну&nbsp;и&nbsp;кому это надо?

## Плюсы способа через lite-youtube-embed
Для улучшения показателей скорости загрузки страниц, придумали lite-youtube-embed, который визуально выглядит как стандартное решение, но&nbsp;загружается примерно в&nbsp;224 раз быстрее и&nbsp;имеет ряд преимуществ:

- Скрипты весят меньше&nbsp;&mdash; не&nbsp;более 16&nbsp;Кбайт!
- Возможность добавить кастомную обложку для видео
- Скрипты загружаются не&nbsp;сразу
- Возможность указывать разные параметры для управления поведением и&nbsp;внешним видом плеера
