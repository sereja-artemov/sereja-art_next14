## Минусы стандартного способа через iframe
Стандартные embed-скрипты для YouTube очень тяжелые и&nbsp;весят сотни килобайт. Весь этот ком скриптов загружается сразу и&nbsp;сильно тормозит загрузку страницы из-за этого снижается самооценка страницы в&nbsp;Google PageSpeed Insights.

## Плюсы способа через lite-youtube-embed
Для поддержания высокой самооценки страниц, придумали lite-youtube-embed, который визуально выглядит как стандартное решение, но&nbsp;загружается примерно в&nbsp;224 раз быстрее и&nbsp;имеет ряд преимуществ:

- Скрипты весят меньше&nbsp;&mdash; не&nbsp;более 16&nbsp;Кбайт!
- Можем поставить картинку для обложки видео
- Скрипты загружаются не&nbsp;сразу
- Можем указывать разные параметры для управления поведением и&nbsp;внешним видом
