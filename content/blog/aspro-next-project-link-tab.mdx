Проблема: при клике на ссылку проекта из выпадающего меню, мы всегда попадаем на таб "Все проекты", а хотим попадать на соответствующий ссылке таб. То есть, если кликнули на ссылку "Роботизация", то и попасть должны на таб "Роботизация".

<Figcaption src="http://tech.sereja-art.ru/upload/blogs/aspro-next-project-link-tab/aspro-next-project-link-tab.png" alt="" />

<CodeTitle title='Костыль для ссылок проектов в URL' lang="js" />
  ```html
    <script>
      $(document).ready(function() {
        var filter = window.location.hash.replace('#', '');
        $('span[data-filter="' + filter + '"]').click();
      });
    </script>
  ```
